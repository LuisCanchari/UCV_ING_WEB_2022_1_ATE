Unidad 1. Introducción a HTML

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

Anatomía de una etiqueta HTML

Para estructurar una pagina web se utiliza un conjunto de etiquetas HTML, las partes basicas de una etique son:

  1. Etiqueta de apertura
  2. Etiqueta de cierre
  3. Contenido
  4. Atributos
